# ホタルイカ予測サイト

富山湾周辺の風向き・気温・天気をまとめて確認できるWebアプリです。ホタルイカの漁や観光船の参考にどうぞ。

## 機能

- **地図**: 富山県周辺にフォーカスしたダークマップ（Leaflet + CARTO）
- **風向き**: Windyのように、富山湾上のグリッドで風向き矢印を表示（矢印の先が風の吹く方向）
- **港**: 滑川・魚津・新湊・岩瀬・四方・入善など主要港をマーカー表示
- **天気**: 各港の現在の気温・天気・風速を一覧し、選択した港の詳細を右パネルに表示
- **データ**: [Open-Meteo](https://open-meteo.com/)（APIキー不要・無料）で取得

## 使い方

1. `index.html` をブラウザで開く（ローカルでOK、CORSはOpen-Meteoが許可しているため問題なし）
2. 左の港リストで港を選ぶか、地図上の⚓マーカーをクリック
3. 選択した港の「現在の天気」「気温」「風向き・風速」を確認
4. 地図上の青い矢印で富山湾上の風向きを確認

## 技術

- フロントのみ（HTML + CSS + JavaScript）
- Leaflet 1.9.4（地図）
- Open-Meteo Forecast API（気象データ）
- 日本時間（Asia/Tokyo）で表示

## ファイル

- `index.html` … 単一ファイルで完結（地図・港・風・天気の表示とAPI呼び出し）

## 注意

- 天気・風は予報データです。実際の漁や運航は各港・船会社の情報を優先してください。
- Open-Meteoは非商用は無料。利用規約は [Open-Meteo](https://open-meteo.com/) を参照してください。
